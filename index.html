REALEDIT: Reddit Edits As a Large-scale Empirical Dataset for Image Transformations
Peter Sushko, Ayana Bharadwaj, Zhi Yang Lim, Vasily Ilin, Ben Caffee, Dongping Chen, Mohammadreza Salehi, Cheng-Yu Hsieh, Ranjay Krishna
Under review
project page / arXiv / code

We introduce REALEDIT, a large-scale image editing dataset with authentic user requests and human-made edits from Reddit, enabling models to better address real-world needs. Our REALEDIT model outperforms competitors significantly and shows potential for broader applications like deepfake detection.
